## What is the command line interface (CLI)?
The Command Line Interface (CLI) is a text-based way to interact with your computer.
## How do I access the command line interface (CLI)?
Press Ctrl + Alt + T (opens a terminal emulator).
## What is a virtual console?
A virtual console (TTY) is a text-only session that runs directly on your computer without a graphical interface.
## What is a terminal emulator?
A terminal emulator is a program inside your graphical desktop that behaves like a virtual console.
## What is bash?

## What is the shell prompt?

## clear
* **definition**:Clears the terminal screen.
* **Usage**:clear
* **Example**:`clear`,`clear && echo "Screen cleared!"`

## echo
* **definition**:Displays text or variables.
* **Usage**:echo [text]
* **Example**:`echo Hello Linux`,`echo $HOME`

## date
* **definition**:Shows or sets system date and time.
* **Usage**:date
* **Example**:`date`,`date +"%A, %d %B %Y"`

## free
* **definition**:Shows memory usage.
* **Usage**:free [options]
* **Example**:`free`,`-h`

## uname
* **definition**:Displays system information.
* **Usage**:uname [options]
* **Example**:`uname`.`-a`

## history
* **definition**:Shows command history.
* **Usage**:history
* **Example**:`history`,`history | tail -10`
## man
* **definition**:Opens manual page for a command.
* **Usage**:man [command]
* **Example**:`man ls`,`man grep`
## tldr
* **definition**:Shows simplified help with examples.
* **Usage**:tldr [command]
* **Example**:`tldr git`,`tldr tar`
## cheat
* **definition**:Displays cheat sheets for commands.
* **Usage**:cheat [command]
* **Example**:`cheat docker`,`cheat awk`
## hostname
* **Definition**:Shows or sets system hostname.
* **Usage**:hostname
* **Example**:`hostname`,`hostname -I`
## df
* **Definition**:Shows disk space usage.
* **Usage**:df [options]
* **Example**:`df`,`df -h`
## du
* **Definition**:Shows directory/file disk usage.
* **Usage**:du [options] [file/directory]
* **Example**:`du -h`,`du -sh Downloads`
## figlet
* **Definition**:Displays large ASCII text.
* **Usage**:figlet [text]
* **Example**:`figlet Linux`,`figlet "Hello World"`
